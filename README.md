# AIGalaxy
使用.NET MAUI BLAZOR  进行 Windows、Android  开发
实现 AI  绘画、AI  聊天功能
可将绘画作品下载到本地
可打包为 Msix和 aab或 apk


将Service中的SignalR的WebsocketUrl替换为你的服务器IP；

可以使用我的MudBlazorServerMinimalAPI(https://github.com/HackerQWERT/MudBlazorServerMinimalAPI) 作为服务器；
